Overview

A terminal-based music player with the following features:

  • Play/Pause/Stop controls
  • Volume adjustment
  • Queue system
  • Playlist navigation
  • Time tracking

Requirements

Python 3.6+
pygame
curses (included in standard library)

Installation

pip install pygame

Usage

python music_playerv3.py
